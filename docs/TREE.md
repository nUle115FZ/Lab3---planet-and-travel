## 🎯 Навигация по проекту

### Хочу понять структуры данных:
```
1. include/core/Sequence.h          ← Начните здесь
2. include/core/DynamicArray.h      ← Потом это
3. include/core/PriorityQueue.h     ← Самое важное! (M-3.4)
```

### Хочу понять граф и алгоритмы:
```
1. include/graph/StarGraph.h        ← Структура графа (M-1.1)
2. include/graph/Dijkstra.h         ← Алгоритм (M-2.2.1)
3. include/graph/Analytics.h        ← Тестирование
```

### Хочу понять интерфейс:
```
1. src/ui/main.cpp                  ← Точка входа
2. include/ui/mainwindow.h          ← Главное окно

4. include/ui/graphview.h           ← Визуализация
5. src/ui/graphview.cpp             ← Рисование
```

### Хочу запустить проект:
