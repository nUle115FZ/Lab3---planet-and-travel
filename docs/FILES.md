# Файлы проекта "Космическая Логистика"

## Структура данных (Backend)

### 1. Sequence.h
**Назначение:** Абстрактный класс-интерфейс последовательности  
**Содержит:**
- Виртуальные методы для работы с коллекциями
- Get, GetFirst, GetLast, GetSize
- Append, Prepend, InsertAt, RemoveAt
- Set, Clear, IsEmpty

### 2. DynamicArray.h
**Назначение:** Динамический массив с автоматическим расширением  
**Реализует:** Sequence<T>  
**Особенности:**
- Автоматическое увеличение capacity при переполнении
- Поддержка оператора []
- Конструктор копирования и оператор присваивания

### 3. PriorityQueue.h
**Назначение:** Очередь с приоритетом (min-heap)  
**Наследует:** DynamicArray<pair<T, K>> (приватно)  
**Методы:**
- Enqueue(item, priority) - добавление с приоритетом
- Dequeue() - извлечение элемента с наивысшим приоритетом
- Peek(), PeekPriority() - просмотр без извлечения
- heapifyUp(), heapifyDown() - поддержка свойства кучи

### 4. StarGraph.h
**Назначение:** Ориентированный взвешенный граф звездных систем  
**Структуры:**
- EdgeWeight - вес ребра (distance + riskFactor)
- Edge - ребро графа (to, weight)
**Методы:**
- AddVertex(name) - добавление планеты
- AddEdge(from, to, weight) - добавление перехода
- GetEdges(vertex) - получение смежных вершин
- GetVertexIndex(name), GetVertexName(id) - конвертация имен

### 5. Dijkstra.h
**Назначение:** Алгоритм поиска кратчайших путей  
**Структуры:**
- DijkstraResult - результат поиска (путь, стоимость)
**Методы:**
- FindShortestPath(start, end) - поиск одного пути
- FindAllShortestPaths(start) - поиск всех путей от вершины
- reconstructPath() - восстановление пути из предшественников

### 6. Analytics.h
**Назначение:** Аналитика и тестирование производительности  
**Структуры:**
- PerformanceResult - результат теста (время, размер графа)
**Методы:**
- measurePerformance() - замер времени выполнения
- runPerformanceTests() - серия тестов
- exportToCSV() - экспорт результатов
- generateDemoGraph() - создание демо-графа
- generateRandomGraph() - генерация случайного графа

## Qt интерфейс (Frontend)

### 7-8. mainwindow.h / mainwindow.cpp
**Назначение:** Главное окно приложения  
**Наследует:** QMainWindow  
**Компоненты:**
- Меню и панель инструментов
- Статус бар
- Интеграция с GraphView
**Слоты:**
- onAddPlanet() - диалог добавления планеты
- onAddEdge() - диалог добавления перехода
- onFindPath() - поиск оптимального маршрута
- onLoadDemo() - загрузка демо-графа
- onRunPerformanceTest() - запуск тестов
- onClearGraph() - очистка графа

### 9. mainwindow.ui
**Назначение:** XML описание интерфейса главного окна  
**Формат:** Qt Designer UI файл

**Наследует:** QWidget

- drawEdge() - рисование перехода со стрелкой
- highlightPath() - подсветка найденного пути
- addLogMessage() - добавление сообщения в лог

## Ключевые моменты для защиты

1. **PriorityQueue наследуется приватно** - чтобы скрыть методы Sequence
2. **Алгоритм Дейкстры имеет сложность O((V+E)log V)** - благодаря PriorityQueue
3. **Граф представлен списком смежности** - эффективно для разреженных графов
4. **Вес ребра учитывает риск** - формула: distance × (1 + riskFactor)
5. **Интерфейс интерактивный** - drag & drop, визуализация пути
6. **Система тестирования встроена** - измерение времени, экспорт в CSV
